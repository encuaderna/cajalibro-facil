# CajaLibro Facil

**Calculadora tecnica para diseno de cajas de libros** — herramienta precisa para dimensionar cajas protectoras artesanales de forma rapida y sin errores.

## Que hace esta app

- Calcula las medidas exactas para construir cajas de libros (estuches, cajas clamshell, etc.)
- Solo ingresas las medidas de tu libro y la app genera todos los cortes
- Incluye opciones de accesibilidad: fuente para dislexia, alto contraste, modo enfoque
- Ideal para principiantes y encuadernadores con experiencia

## Como usarla

Haz clic en el siguiente enlace y empieza directamente en el navegador, sin instalar nada:

**Abrir app:** https://box-craft-logic.base44.app

## Para quien es

Para quienes encuadernan artesanalmente y quieren construir cajas protectoras para sus libros sin calcular a mano.
